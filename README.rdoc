= brou-scrapper

Utilidad que corre en background (cron), que chequea cambios en tu cuenta bancaria BROU y te notifica a tu cuenta Gmail.

== Uso

Luego de configurar appropiadamente config.yml. Se puede utilizar de la siguiente forma, dentro de cron:

0-59/10 * * * * cd /path/to/brou-scrapper && /path/to/bundle exec /path/to/ruby /path/to/brou-scrapper/bin/scrapper.rb
